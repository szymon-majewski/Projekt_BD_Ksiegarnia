# Projekt_BD_Ksiegarnia

Projekt bazy danych księgarni internetowej, obsługującej podstawowe zadania konieczne do poprawnego funkcjonowania – zarówno od strony pracownika jak i klienta, wykonany na potrzeby laboratoriów z przedmiotu "Bazy Danych" na Uniwersytecie Jagiellońskim.
Jest to ramowa forma bazy danych, pozostawiająca duże pole do rozwoju, zgodne z wybranym kierunkiem oraz dopasowane do odpowiednich przyszłych potrzeb.
